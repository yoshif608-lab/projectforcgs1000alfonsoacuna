Emerging Technology

Technology: Augmented Analytics (Generative AI in Data Analytics)

What it is: Analytics applies machine learning and generative AI to automate data preparation and explain results in plain language, rather than requiring an analyst to write every query by hand.

How it's used: Analysts use AI "copilots" built into BI platforms to ask questions in natural language, auto-generate first-draft dashboards, and flag anomalies without manual digging.

How I'll think I'll use it: As I move from cybersecurity into data analysis, I expect to use these AI tools to speed up so I can focus more time on interpreting results and communicating them like it is.

Why it interests me: It sits right at the intersection of my two interests, mostly because using automated, security-aware tools to work faster and more accurately with data is exactly where the field is heading.
